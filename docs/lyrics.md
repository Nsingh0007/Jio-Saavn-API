# Lyrics

## Song lyrics by song ID

> Lyrics are only available for a limited set of songs.

+++ Request

HTTP

```bash
https://saavn.dev/lyrics?id=mPTrDSun
```

CURL

```bash
curl -X GET 'https://saavn.dev/lyrics?id=mPTrDSun' \
 -H 'content-type: application/json'
```
