# Search Playlists

Search playlists by query.

+++ Request

HTTP

```bash
https://saavn.dev/search/playlists?query=sia
```

CURL

```bash
curl -X GET 'https://saavn.dev/search/playlists?query=sia' \
 -H 'content-type: application/json'
```
