# Search All

Search songs, albums, playlists, artists, etc with a query.

+++ Request

HTTP

```bash
https://saavn.dev/search/all?query=imagine+dragons
```

CURL

```bash
curl -X GET 'https://saavn.dev/search/all?query=imagine+dragons' \
 -H 'content-type: application/json'
```
