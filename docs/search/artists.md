# Search Artists

Search artists by query.

+++ Request

HTTP

```bash
https://saavn.dev/search/artists?query=imagine+dragons
```

CURL

```bash
curl -X GET 'https://saavn.dev/search/artists?query=imagine+dragons' \
 -H 'content-type: application/json'
```
